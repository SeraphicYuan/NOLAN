# Video Essay Style Asset Checklist

A comprehensive list of visual assets needed for each video essay style. Use this to ensure consistent, professional-quality graphics across all styles.

**Sources:** Research from [Visme](https://visme.co/blog/best-infographic-tools/), [Envato Elements](https://elements.envato.com/video-templates/documentary), [Motion Array](https://motionarray.com/motion-graphics-templates/lower-thirds/), [Storyblocks](https://www.storyblocks.com/resources/blog/the-art-of-the-video-essay), [FilterGrade](https://filtergrade.com/15-lower-thirds-templates-for-documentaries/)

---

## Asset Categories Overview

| Category | Priority | Description |
|----------|----------|-------------|
| **Title & Text** | Critical | Opening titles, section headers, quote cards |
| **Lower Thirds** | Critical | Speaker IDs, location tags, source citations |
| **Data Visualization** | High | Charts, graphs, statistics, counters |
| **Callouts & Annotations** | High | Arrows, highlights, labels, pointers |
| **Transitions & Dividers** | Medium | Section breaks, chapter markers, wipes |
| **Frames & Mockups** | Medium | Screen frames, device mockups, photo frames |
| **Maps & Geography** | Medium | Location maps, route animations, region highlights |
| **Comparison & Layout** | Medium | Split screens, before/after, side-by-side |
| **Progress & Timeline** | Medium | Timelines, roadmaps, step indicators |
| **Shapes & Decorative** | Low | Background textures, accent shapes, dividers |

## Style-Specific Considerations

Each style should have its own visual treatment of these assets:

| Style | Color Palette | Typography | Texture | Animation |
|-------|---------------|------------|---------|-----------|
| **noir-essay** | Dark grays, white accents | Serif, high contrast | Film grain, vignette | Smooth fades |
| **cold-data** | Blues, grays, minimal color | Sans-serif, clean | None/subtle | Sharp, precise |
| **modern-creator** | Vibrant, high saturation | Bold sans-serif | Gradients | Bouncy, energetic |
| **academic-paper** | Muted, professional | Serif body, sans headers | Paper texture | Minimal |
| **documentary** | Earth tones, cinematic | Mixed serif/sans | Film grain | Cinematic |
| **podcast-visual** | Brand colors, warm | Friendly sans-serif | Soft gradients | Gentle |
| **retro-synthwave** | Neon pink/cyan/purple | Display fonts | Scan lines, glow | Pulsing, retro |
| **breaking-news** | Red, white, blue | Bold condensed | Metallic | Urgent, ticker |
| **minimalist-white** | White, single accent | Thin sans-serif | None | Subtle |
| **true-crime** | Dark, desaturated | Typewriter, distressed | Grain, scratches | Tense |
| **nature-documentary** | Greens, earth tones | Elegant serif | Organic textures | Slow, majestic |

## Implementation Priority

### Phase 1: Core (MVP)
- Title cards (main, chapter)
- Lower thirds (name, location, source)
- Quote cards
- Basic charts (bar, line, pie)
- Stat counter
- Arrows and highlights

### Phase 2: Enhanced
- All chart types
- Device mockups (browser, phone)
- Timeline and process diagrams
- Split screen layouts
- Map markers and routes

### Phase 3: Complete
- Social media frames
- Full flowchart system
- Grid layouts
- Decorative elements
- Style-specific variants

## File Naming Convention

```
{category}-{type}[-{variant}].svg

Examples:
  lower-third-name.svg
  lower-third-name-two-line.svg
  chart-bar-horizontal.svg
  frame-browser-chrome.svg
```

## Folder Structure

```
assets/
├── common/
│   ├── icons/           # Semantic icons (check, x, warning, etc.)
│   ├── shapes/          # Basic shapes (arrows, highlights)
│   └── ui/              # UI elements (progress bars, dividers)
└── styles/
    ├── noir-essay/
    │   ├── titles/      # Title cards
    │   ├── lower-thirds/
    │   ├── charts/
    │   ├── callouts/
    │   ├── frames/
    │   └── decorative/

    │   └── ...

        └── ...
```
