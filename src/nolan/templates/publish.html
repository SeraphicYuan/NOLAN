Publish a Beautiful Article

Turn a URL, a document path, or pasted text into a self-contained, offline, single-file HTML article — the second NOLAN output medium alongside video. An agent writes and builds it using the beautiful-article skill (reacticle themes + figure library). Authoring runs a Claude agent and takes a few minutes.

Output lands in projects/_published/<slug>/article/article.html — fully offline, openable and shareable as a single file.
