## Clothing — no channel-identity content copied verbatim

- **Sponsor read + coded URL (`ground.news/stickman`)** → **SKIPPED entirely.** Not written. (draft-01 included it; draft-02 omits it per the task's hard rule — this is the main deliberate divergence.)
- **Persona label ("I'm literally a stickman")** → **ADAPTED, not claimed.** Rendered as the neutral, same-function line "I'm just some guy who read the permit filings so you don't have to" (self-deprecation that disarms before a polarizing take).
- **Catchphrase sign-off ("That's all I've got for today's video…"), channel-members thanks, Substack/newsletter plug** → **ADAPTED to a neutral functional equivalent.** The close keeps the *functions* (warm wrap, genuine like/subscribe, "sources are in the description," invite-disagreement) without the verbatim catchphrase or the members/Substack furniture. "take care" is retained as a generic, non-exclusive sign-off, not a channel-only phrase.
- **Exemplar Lines** → used only as a **cadence tuning-fork** ("Picture this."; "they're both right, to some degree.") — no political exemplar line transplanted.
